4,Hexo免输入密码部署到github
加入SSH Agent
ssh-agent -s
如果出现类似Agent pid XXXX这样的字样，则跳过下一步，否则输入：
eval ssh-agent -s
直到出现Agent pid XXXX这样的提示之后，输入：
ssh-add ~/.ssh/id_rsa

执行hexo d 时，如果更换目录需要重新添加一下代理
